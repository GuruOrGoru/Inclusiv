Accessibility Help

Inclusiv is designed to be accessible to everyone. Here's how to make the most of our platform using various assistive technologies and accessibility features.

Keyboard Navigation

  • Tab Navigate forward through interactive elements
  • Shift + Tab Navigate backward through interactive elements
  • Enter or Space Activate buttons and links
  • Escape Close modals and menus

Screen Reader Support

Our platform is fully compatible with screen readers including NVDA, JAWS, VoiceOver, and TalkBack. We use semantic HTML and ARIA labels to ensure all content is properly announced.

Screen Reader Tips:

  • Use landmark navigation to jump between main sections
  • Headings are properly structured for easy navigation
  • All form inputs have descriptive labels
  • Job listings include detailed descriptions for context

Visual Accessibility

High Contrast

Our dark theme provides excellent contrast and meets WCAG AA standards for color contrast ratios.

Font Size

You can increase text size using your browser's zoom function (Ctrl/Cmd + Plus) up to 200% without losing functionality.

Reduced Motion

If you have motion sensitivity, our platform respects your system's reduced motion preferences.

Voice Control & Motor Accessibility

  • Compatible with voice control software like Dragon NaturallySpeaking
  • Large click targets (minimum 44px) for easier interaction
  • No time limits on form completion
  • Generous spacing between interactive elements

Cognitive Accessibility

  • Clear, simple language throughout the platform
  • Consistent navigation and layout patterns
  • Auto-save functionality for forms
  • Search highlighting to help locate relevant content

Need Additional Help?

If you're experiencing any accessibility issues or need additional accommodations, please don't hesitate to contact us. We're committed to ensuring everyone can use our platform effectively.
